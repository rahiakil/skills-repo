# Photonic Design Agent Skills

A curated, evolving library of Claude skills purpose-built for photonic design research,
academic writing, data analytics, and investor communication.

**Maintained by:** Parag
**Updated:** Daily via automated skill discovery
**Goal:** Build a fully stable, agentic photonic design system — from research ingestion
to publication to investor pitch.

---

## Skill Index

| Skill | Purpose |
|-------|---------|
| `deep-research` | 13-agent research pipeline — literature, PRISMA, fact-check |
| `academic-paper` | Full paper drafting, LaTeX, citation management |
| `paper-reviewer` | Peer review simulation, quality rubrics |
| `scientific-writing` | IMRAD structure, figures, journal formatting |
| `photonics-research` | Domain-specific photonics data ingestion, simulation analysis |
| `data-analytics` | Python analytics scripts, EDA, publication-quality charts |
| `knowledge-network` | Interlink papers and notes into searchable knowledge graphs |
| `article-extractor` | Pull clean text from any web article/paper URL |
| `youtube-transcript` | Transcribe conference talks and lectures |
| `pitch-deck` | Investor-ready slide decks, VC-tailored messaging |
| `blog-writer` | Research-to-blog pipeline, technical and accessible tone |
| `code-explainer` | Annotate and explain simulation/analysis code |

---

## How Skills Are Updated

A daily automated task researches GitHub and the web for new skills relevant to:
- Photonics / optics / nanophotonics
- Academic research pipelines
- Data analytics and visualization
- Investor communication and pitch decks
- Information summarization and knowledge funneling

New skills are committed here. You review and push to your photonic design repo.

---

## Skill Format

Every skill follows the `SKILL.md` standard (compatible with Claude Code, Cowork, Cursor,
Codex CLI, and Gemini CLI). Each folder contains a single `SKILL.md` with YAML frontmatter
and instructions.
